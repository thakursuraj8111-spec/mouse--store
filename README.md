# 🖱️ Zebronics Mouse Store (Frontend Project)

A modern and responsive e-commerce frontend website built using **HTML, CSS, and JavaScript**.  
This project simulates a basic online mouse store with cart and search functionality.

---

## 🚀 Features

- 🛒 Add to Cart system
- 💰 Live total price calculation
- 🔍 Real-time search filter
- ❌ Remove items from cart
- 📱 Responsive design (mobile friendly)
- 🎨 Modern dark UI design

---

## 🧑‍💻 Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

## 📸 Preview

> This project includes:
- Product listing cards
- Shopping cart sidebar
- Search functionality
- Toast notification system

---

## 🎯 Purpose

This project was created for **learning web development and UI design skills**.  
It helps understand how real e-commerce websites work at the frontend level.
